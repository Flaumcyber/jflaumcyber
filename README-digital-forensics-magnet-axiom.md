# Digital Forensics — Magnet AXIOM Project

A digital forensic investigation conducted using Magnet Forensics (AXIOM) to analyze a computer involved in a criminal case — examining browser history, emails, downloads, user accounts, passwords, media files, and cloud artifacts to answer investigative questions and document findings in a forensic report.

## What I Learned (and Why It Matters)

I learned how digital forensic tools can recover and organize evidence from a computer while maintaining the integrity of the investigation. I gained experience locating user activity, internet history, downloaded files, email evidence, and media artifacts, then documenting how each piece of evidence supported the investigation. These skills are important because digital forensics plays a critical role in criminal investigations by uncovering reliable digital evidence that can be used in court.

## What I Enjoyed Most

I enjoyed working through the investigation like a real forensic examiner. Searching through digital artifacts, answering investigative questions, and connecting evidence to build a timeline made the project engaging and demonstrated how technical skills are used to solve real-world cases.
