A desktop Blackjack game built in Python with Tkinter, themed after the Lucky 38 Casino from Fallout: New Vegas. Full deck and shuffle logic, Ace-aware scoring, an automatic dealer, and running win/loss statistics.

The version below is a browser-playable recreation of the same rules and scoring logic — built in JavaScript since Tkinter can't run in a browser — so you can actually sit down at the table. The original Python source is further down, along with a full walkthrough of how it was built.

// how it works

Step by step, here's how the original Python/Tkinter version was put together:

  1. 1. Set up the Tkinter window and layout grid Created the main tkinter.Tk() window, sized it, gave it a black background to sell the casino look, and laid out a column grid so the dealer area, player area, and buttons would all line up predictably regardless of window size.
  2. 2. Load the card images load_images() loops through all four suits and, for each one, loads number cards 1–10 plus the three face cards (jack, queen, king) as PhotoImage objects, storing each as a (value, image) tuple. Face cards are all stored with a value of 10, matching standard Blackjack rules.
  3. 3. Build and shuffle the deck The loaded card tuples get combined into a single deck list and shuffled with random.shuffle(). _deal_card() pops a card off the top, immediately appends it back to the bottom, and drops the image into a Label widget so it renders in the UI — the same list doubles as both the shuffled shoe and the render queue.
  4. 4. Write the scoring logic score_hand() is the trickiest part: it walks each card in a hand, and only the first Ace encountered is allowed to count as 11 instead of 1. If that would bust the hand (score over 21), it subtracts 10 back off and locks the Ace flag so it can't be re-used. Every other Ace just counts as 1.
  5. 5. Wire up Hit and Stand deal_player() (bound to the Hit button) deals one card to the player and re-checks the score for a bust. deal_dealer() (bound to Stand) plays the dealer's hand out automatically — hitting on anything below 17 — then compares both scores to decide the winner and updates the running games_played / games_won / games_lost counters.
  6. 6. Handle New Game resets Rather than tracking and destroying individual card Label widgets one by one, new_game() just destroys the entire dealer_card_frame / player_card_frame container and rebuilds a fresh empty one — the simplest way to clear a hand's worth of images in one call.
  7. 7. Surface game statistics Instead of a separate stats panel, update_window_title() writes the running totals straight into the window's title bar after every round, so the stats are always visible without taking up any table space.
Worth noting: the original code builds its shoe from three combined 52-card decks rather than one, and the button labeled "Game Over" is actually wired to a function named shuffle() that shows a stats popup and closes the window — a couple of quirks left in on purpose so the walkthrough matches the real source below.

import random
import tkinter as tk
from tkinter import messagebox

try:
    import tkinter
except ImportError:
    import Tkinter as tkinter


def load_images(card_images):
    suits = ['heart', 'club', 'diamond', 'spade']
    face_cards = ['jack', 'queen', 'king']

    if tkinter.TkVersion >= 8.6:
        extension = 'png'
    else:
        extension = 'ppm'

    # for each suit, retrieve the image for the cards
    for suit in suits:
        # first the number cards 1 to 10
        for card in range(1, 11):
            name = 'cards/{}_{}.{}'.format(str(card), suit, extension)
            image = tkinter.PhotoImage(file=name)
            card_images.append((card, image, ))

        # next the face cards
        for card in face_cards:
            name = 'cards/{}_{}.{}'.format(str(card), suit, extension)
            image = tkinter.PhotoImage(file=name)
            card_images.append((10, image, ))


def _deal_card(frame):
    # pop the next card off the top of the deck
    next_card = deck.pop(0)  # Use 0 to take card from top of deck
    # and add it back to the deck
    deck.append(next_card)
    # add the image to a label and display the label
    tkinter.Label(frame, image=next_card[1], relief="raised").pack(side="left")
    # now return the card's face value
    return next_card


def score_hand(hand):
    # Calculate the total score of all cards in the list
    # Only one ace an have the value 11 and this will be reduce to 1 if the hand would bust
    score = 0
    ace = False
    for next_card in hand:
        card_value = next_card[0]
        if card_value == 1 and not ace:
            ace = True
            card_value = 11
        score += card_value
        # if we would bust, check if there is an ace and subtract 10
        if score > 21 and ace:
            score -= 10
            ace = False
    return score


def deal_dealer():
    global games_played, games_won, games_lost, twenty_ones_scored

    dealer_score = score_hand(dealer_hand)
    while 0 < dealer_score < 17:
        dealer_hand.append(_deal_card(dealer_card_frame))
        dealer_score = score_hand(dealer_hand)
        dealer_score_label.set(dealer_score)

    player_score = score_hand(player_hand)
    if player_score > 21:
        result_text.set("Dealer wins!")
        games_lost += 1
    elif dealer_score > 21 or dealer_score < player_score:
        result_text.set("Player wins!")
        games_won += 1
    elif dealer_score > player_score:
        result_text.set("Dealer wins!")
        games_lost += 1
    else:
        result_text.set("Draw!")

    games_played += 1

    if player_score == 21:
        twenty_ones_scored += 1

    # Update the game statistics in the main window title
    update_window_title()


def deal_player():
    player_hand.append(_deal_card(player_card_frame))
    player_score = score_hand(player_hand)

    player_score_label.set(player_score)
    if player_score > 21:
        result_text.set("Dealer Wins!")


def initial_deal():
    deal_player()
    dealer_hand.append(_deal_card(dealer_card_frame))
    dealer_score_label.set(score_hand(dealer_hand))
    deal_player()
