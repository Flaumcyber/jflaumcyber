Penetration Testing Report
Attack from End User's Desktop
USB Attacks · Spear Phishing · SQL Injection · Privilege Escalation █
// about

A team-led penetration testing project simulating a USB-based attack on a retail corporate network, targeting end-user desktops to demonstrate credential theft, data exfiltration, and the real-world impact of weak endpoint security. As team lead, I coordinated the research, assigned responsibilities, and integrated the team's findings into the final report — covering USB drop attacks, spear phishing, session hijacking, SQL injection, and privilege escalation.

I worked at Costco to help put myself through college, and the idea for this project actually came from the job itself. Spending that much time on the warehouse floor, I noticed blatant security gaps first-hand — unlocked workstations, open USB ports, unattended logins — the kind of everyday negligence that never makes it into a security textbook. That's what pushed me to build this project around a retail environment instead of a hypothetical one: I wanted to show how the vulnerabilities I was actually seeing could realistically be chained together into a full attack.
