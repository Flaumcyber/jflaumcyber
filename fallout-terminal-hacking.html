// about

A fully playable Fallout terminal hacking minigame written in pure Python. It replicates the classic mechanics from Fallout 3, 4, and New Vegas — pick the correct password out of a pool of decoy words within a limited number of attempts, using a "Likeness" score (the number of characters that match the secret word in the exact same position) to narrow down each guess.

step 01
Import modules & set up the configuration
The game only needs three standard-library modules: random to shuffle and pick words, sys to write characters to the terminal one at a time, and time to pace that typing effect. Below that, a WORD_POOL list holds every possible password candidate — all uppercase, all the same length, just like the real Pip-Boy terminals — and ATTEMPTS_ALLOWED caps how many guesses the player gets before the terminal locks.
import random
import sys
import time

# Configuration for game difficulty
WORD_POOL = [
    "REACTOR", "NUCLEAR", "STATION", "WARFARE", "FALLOUT",
    "SHELTER", "MUTANTS", "ROBOTIC", "BUNKERX", "CAPITAL",
    "REMANTS", "CRIMSON", "FACTION", "SURVIVE", "URANIUM"
]
ATTEMPTS_ALLOWED = 4
step 02
Build the retro typewriter effect
To sell the "ancient terminal" feel, text can't just appear instantly. type_effect() loops over every character in a string, writes it directly to stdout with sys.stdout.write(), flushes the buffer so it renders immediately, and then sleeps for a tiny delay before the next character. The result is that classic line-by-line CRT typing animation.
def type_effect(text, delay=0.03):
    """Simulates the iconic retro terminal text rendering."""
    for char in text:
        sys.stdout.write(char)
        sys.stdout.flush()
        time.sleep(delay)
    print()
step 03
Write the Likeness calculator
This is the actual "hacking" logic. get_likeness() pairs up each character of the player's guess with the same-position character of the secret word using zip(), then counts how many positions match exactly. That single number — the Likeness score — is the only feedback the player gets, exactly like in the games.
def get_likeness(word_a, word_b):
    """Calculates matching characters in identical positions."""
    return sum(1 for a, b in zip(word_a, word_b) if a == b)
step 04
Set up the game session
At the start of play_game(), print the RobCo-style header, then use random.sample() to pull 8 unique words out of the pool for this session and random.choice() to pick one of those 8 as the actual secret password. Tracking attempts_left separately from the fixed ATTEMPTS_ALLOWED constant lets the loop count down each round.

Build the main guessing loop
While the player still has attempts, print the remaining count and the full word matrix, then read their guess with input(). Anything not in game_words is rejected without costing an attempt — a nice usability touch. A correct guess ends the game immediately with a success message and returns out of the function entirely.

// terminal_hack.py 
import random
import sys
import time
# Configuration for game difficulty
WORD_POOL = [
    "REACTOR", "NUCLEAR", "STATION", "WARFARE", "FALLOUT", 
    "SHELTER", "MUTANTS", "ROBOTIC", "BUNKERX", "CAPITAL",
    "REMANTS", "CRIMSON", "FACTION", "SURVIVE", "URANIUM"
]
ATTEMPTS_ALLOWED = 4
def type_effect(text, delay=0.03):
    """Simulates the iconic retro terminal text rendering."""
    for char in text:
        sys.stdout.write(char)
        sys.stdout.flush()
        time.sleep(delay)
    print()
def get_likeness(word_a, word_b):
    """Calculates matching characters in identical positions."""
    return sum(1 for a, b in zip(word_a, word_b) if a == b)
def play_game():
    # Setup terminal header
    print("=" * 50)
    type_effect("ROBCO INDUSTRIES (TM) TERMLINK PROTOCOL")
    type_effect("ENTER PASSWORD NOW")
    print("=" * 50)
    
    # Select words for the session
    game_words = random.sample(WORD_POOL, 8)
    secret_word = random.choice(game_words)
    word_length = len(secret_word)
    
    attempts_left = ATTEMPTS_ALLOWED
    # Main game loop
    while attempts_left > 0:
        print(f"\n{attempts_left} ATTEMPT(S) REMAINING.\n")
        
        # Display the candidate password matrix
        for word in game_words:
            print(f" > {word}")
        print()
        
        # User input handling
        guess = input("ENTER GUESS > ").strip().upper()
        
        if guess not in game_words:
            type_effect("ERR: INVALID TERMINAL ENTRY. TRY AGAIN.")
            continue
            
        if guess == secret_word:
            print()
            type_effect("SUCCESS! MATCH FOUND.")
            type_effect("ACCESS GRANTED. OPENING SECURITY DOOR...")
            return
        
        # Incorrect guess sequence
        attempts_left -= 1
        likeness = get_likeness(guess, secret_word)
        
        type_effect("ACCESS DENIED.")
        type_effect(f"LIKENESS={likeness}/{word_length}")
        print("-" * 30)
    # Game over sequence
    print()
    type_effect("TERMINAL LOCKED.")
    type_effect(f"THE CORRECT PASSWORD WAS: {secret_word}")
    type_effect("PLEASE CONTACT YOUR SYSTEM ADMINISTRATOR.")
if __name__ == "__main__":
    play_game()
